Mysoftheaven Inventory with POS
Installation & Updating Steps

    Please open:
  1. Example 1 : http://localhost/inventory/setup
    Example 2 : https://www.example.com/setup
  2. In that form you will get two options. 1. Install & 2. Update
    • Option 1: Select Install button for new installation.
    • Option 2: Select Update button for updating database.
  3. After Updating: Please copy "uploads" folder from your old version of application to newer version of application.
  4. After update process -> Login -> Go to -> Settings -> Database Updates -> Click on Update Database.
    Note: Before updating database please take database backup from your server and make it sure.
